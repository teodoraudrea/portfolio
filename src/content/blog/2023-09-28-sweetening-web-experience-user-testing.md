## Overview
An overview of user testing sessions run during Orientation Week to validate new guides and course-search experiences.

## Key takeaways
- Students found guides easier to navigate and understand.
- Qualitative feedback informed future iterations and rollouts.
- Ongoing testing is critical to maintaining user-focused delivery.
