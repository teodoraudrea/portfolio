## Overview
Developed a management portal for BIAN APIs using NginX for high-performance routing and Golang
for backend services.

## Highlights
- Implemented a list command to surface published APIs and proxies quickly.
- Built a delete command to remove one or multiple API proxies safely.
- Streamlined API discovery and operational workflows for teams managing integrations.

## Outcomes
The portal simplified API lifecycle management and improved visibility across integration assets.
